# Product 2026

Design system and case study explorations for game product.

## Overview

This repository contains design system components, case study explorations, and design iterations for game product development. Organized as a working reference for product design and strategic direction.

## Contents

- **Design System** — Component library, typography, tokens, and visual patterns
- **Case Studies** — Wildland, Magicaland, Fish Game explorations and design decisions
- **Explorations** — Navigation, layout, and feature direction iterations
- **Portfolio versions** — Design evolution and refinement process

## Structure

```
├── Leon Maimon Design System.dc.html
├── Leon Maimon Portfolio v[1-22].dc.html
├── Case Studies/
│   ├── Wildland explorations
│   ├── Magicaland explorations
│   └── Fish Game case study
├── Explorations/
│   ├── Layout variations
│   ├── Navigation patterns
│   └── Feature explorations
└── Assets/
    └── Game diagrams and visual references
```

## Usage

Files are Figma Design Canvas (.dc.html) exports. Open in a browser or Figma to view and iterate.
